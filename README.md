## ✨ Features
* **Instant Extraction:** Click any pixel on an uploaded image to get its color data.
* **Dual Formats:** Get both **HEX** and **RGB** codes simultaneously.
* **One-Click Copy:** Save codes to your clipboard instantly with visual confirmation.
* **Modern UI:** Supports **Dark Mode** and **Light Mode** (saves your preference!).
* **Privacy First:** All image processing happens locally in your browser. No data is ever uploaded to a server.

## 🛠️ Built With
* **HTML5 & CSS3** (Custom properties/Variables)
* **Vanilla JavaScript** (Canvas API)
* **JSON** (For dynamic site configuration)
